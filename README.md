# Robot Soccer

This project teaches children to program arduino [mBot](http://learn.makeblock.com/en/mbot/) C++ and Raspberry Pi Javascript.  In the time of coronavirus, this is set up to be performed remotely using 

[code-server](https://github.com/cdr/code-server)
[code-server image](https://hub.docker.com/r/linuxserver/code-server)
[node-webrtc](https://github.com/node-webrtc/node-webrtc)
[medooze media-server](https://github.com/medooze/media-server)
[Apurv's Blog](https://apurv.me/accessing-and-streaming-webcam-in-nodejs-using-opencv-and-websockets/)
